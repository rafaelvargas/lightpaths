# Lightpaths

A simple ray-tracer made with Rust

## Running
When running for the first time, [Cargo](https://doc.rust-lang.org/cargo/guide/) will install all the package dependencies automatically.

### Optimized
```bash
cargo run --release
```
### Debug
```bash
cargo run
```
